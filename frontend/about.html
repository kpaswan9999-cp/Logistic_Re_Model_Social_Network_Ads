SocialFlow is a high-performance machine learning inference dashboard that predicts consumer purchase behavior from social network advertisement data. By leveraging statistical modeling, we deliver actionable conversion probability scores to businesses.

CORE LOGIC

  • σ
    Logistic Regression: Binary classifier using sigmoid activation to map inputs to a purchase probability between 0 and 1.
  • ∑
    Standard Scaling: Feature normalization ensures equal weight between Age (18–60) and Salary (₹50K–₹1Cr) in the model.
  • ⊘
    Median Imputation: Null values in the dataset are replaced with column medians to prevent statistical drift during training.
